Available Operations

+ Addition Adds two numbers
− Subtraction Subtracts second from first
× Multiplication Multiplies two numbers
÷ Division Divides first by second
C/CE Clear Functions Clear all or clear entry
← Backspace Delete last digit
Keyboard support enabled

How to Use

  • Click buttons or use keyboard for input
  • C : Clear all | CE : Clear entry
  • Backspace : Delete last digit
  • Press Enter or = for result
  • Press Escape to clear everything
  • Use . for decimal numbers
  • Keyboard numbers 0-9 work directly
  • Press operators (+, -, *, /) on keyboard

Keyboard Shortcuts

Enter = (Equals)
Esc Clear All
Del Clear Entry
